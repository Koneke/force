# Game structure

- Draw 7, shuffle 2 back
- Each player has 15 life

# Turn structure

- Reset (untap)
- Generate (4 energy per turn, except 2 on turn 1)
- Draw (1 per turn)
- Main/battle
- End

# Combat

- Must attack exhausted if any
- If no exhausted, go face
- Attacking exhausts
- *Blocker*s can intercept _ONE_ attack per turn, and it doesn't exhaust. They have to be rested.
